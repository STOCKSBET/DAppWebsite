Open Source Разработка

Этот контракт позволяет делать ставки на цены акций NASDAQ и выигрывать у игроков, которые поставили против Вас.

Контракт контролирует средства игроков, рассчитывает изменение цены акций между ценой закрытия и ценой открытия рынка плюс одна минута. Контракт фиксирует вознаграждения для победивших игроков. Цены на акции берутся из внешнего источника https://www.alphavantage.co в качестве основного API и Stocks.Bet API в качестве альтернативы. Эти цены определяются с помощью расширенной сети оракулов Oraclize.it в начале (16:30 Нью-Йорк) и в конце периода ставок (10:00 Нью-Йорк).

В Beta версии приложения используются цены следующих компаний:

  • Microsoft (MSFT),
  • Google (GOOG),
  • Apple (AAPL).

Игроки могут делать ставки на котируемые акции, цены на которые, по их мнению, будут расти больше по сравнению с другими акциями. Чтобы разместить ставку и претендовать на награду игроки должны использовать плагин Metamask для браузеров Chrome / Firefox / Opera / Brave.

Все операции полностью защищены Блокчейном. Никто не имеет контроля над инвестициями игроков или процессом изменения цены акций.
